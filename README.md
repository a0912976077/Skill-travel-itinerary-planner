# 🧭 旅遊行程規劃器
**AI-Powered Travel Itinerary Planner**

---

## 📖 專案簡介

這是一個以 AI 驅動的旅遊行程規劃工具，使用者只需輸入目的地、天數、旅遊風格等參數，即可自動生成一份結構完整的個人化旅遊行程。

本工具源自我個人的旅遊規劃工作流程——每次出發前都需要手動查詢景點開放時間、規劃路線順序、計算 Pass 卡效益，耗時且容易遺漏細節。因此將這套流程封裝成 AI Skill，讓規劃更快速、一致且可重複使用。

---

## ✨ 功能特色

- 🗺️ **智慧行程生成** — 輸入目的地與天數，AI 自動規劃每日行程
- 📍 **景點地理群聚** — 依地理位置集中安排，避免來回奔波
- 🎫 **Pass 卡優化建議** — 自動計算持有票卡的最佳使用方式
- 🌤️ **備用方案規劃** — 每天提供雨天或臨時變動的備用選項
- 🍜 **多元興趣標籤** — 美食、文化、購物、自然等多種偏好可選
- ⚡ **步調滑桿調整** — 從悠閒到緊湊，自由調整行程密度
- 📱 **手機友善介面** — 響應式設計，手機平板皆可使用

---

## 🛠️ 使用方式

1. 開啟網頁：[https://a0912976077.github.io/travel-planner/](https://a0912976077.github.io/travel-planner/)
2. 填入目的地城市與住宿區域
3. 設定旅行天數、人數與月份
4. 選擇旅遊興趣標籤（可多選）
5. 調整步調滑桿與預算等級
6. 選填 Tourist Pass 或特殊需求
7. 按下「✦ 產生我的專屬行程」
8. 等待 AI 生成完整行程
